## 本地运行

如果你想在本地运行本站，可以按照以下步骤操作：

```bash
# 克隆仓库
git clone https://github.com/Super-Dian/Personal_Blog.git
cd Personal_Blog

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

启动后，在浏览器中打开终端显示的地址即可预览站点。
